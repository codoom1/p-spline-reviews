Additional Resources
------------------
- Most papers can be accessed through:
  * University library subscriptions

- For PDF access:
  * Check the /assets/papers/ directory in the workspace
  * Use DOI links when available
  * Contact authors directly if needed
  * Save all PDFs in the assets/papers/ directory using the format: authorYYYY.pdf
    (e.g., eilers1996.pdf, kong2023.pdf)
  * For multiple authors, use first author's last name
  * Keep filenames lowercase for consistency
